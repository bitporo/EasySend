# easy-send

#### 介绍
简传，可在局域网环境内共享文本及文件的小软件，采用web技术，仅需一台主机电脑启动，需要获取共享的电脑只需使用浏览器访问即可

#### 软件架构
采用electron-egg + uniapp 实现
